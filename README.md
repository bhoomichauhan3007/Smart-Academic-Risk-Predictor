# Smart Academic Risk Predictor

## Overview
Smart Academic Risk Predictor is a machine learning based system that identifies students who are at risk of poor academic performance.

## Features
- Student login system
- Academic questionnaire
- Machine learning risk prediction
- Personalized recommendations
- Final performance analysis

## Technologies Used
- Python
- Machine Learning
- Pandas
- Scikit-learn


## Project Structure
- Login Module
- Dashboard
- Academic Questions
- Risk Prediction Model
- Recommendation System
- Final Result Analysis
- For future stress detection through facial expression.

## Dataset
The project uses a student academic dataset containing features such as attendance, study habits, and past performance.

## How to Run
1. Install dependencies  
2. Run App.py  
3. Follow the dashboard instructions
